Features

  • Register, sign in with email or sign in with Google
  • Add tasks with title, description and due date
  • Tasks are grouped into "ongoing", "completed", "expired"
  • Tasks can be edited, deleted, changed from "ongoing" to "completed" and vice versa
  • Show current weather based on location (location access is required)
  • Get 5-Day (3-Hour Step) weather forecast
  • Get random inspiring quotes
  • Secure authentication using access token stored in memory, with 15-minute validity, csrf token and refresh token, with 1-day validity
  • Responsive design enables this website to adapt to the size of the screen

P.S.

The frontend, backend and database of this project use free-tier hosting so this website might not always be accessible and a significant initial loading time is expected.
